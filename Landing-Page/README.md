# Specialized Dashboard Landing Page

## Development

The landing page can be viewed locally during development.

### Running the Server

To serve the landing page, run the following command in the `/home/GUI` directory:

```bash
python3 -m http.server 8085
```

### Viewing the Page

Once the server is running, you can view the landing page at:
[http://localhost:8085/Landing-Page/index.html](http://localhost:8085/Landing-Page/index.html)

### Installer Link

The download button points to `../Electron-Dashboard/release/System Dashboard Setup 1.0.0.exe`. careful directory structure is required for this link to work locally.
